# 数据视界

这是一个按照仓库内 `需求文档` 生成的前端网站原型，基于 React + Vite 实现。

## 快速开始

1. 安装依赖

```bash
npm install
```

2. 启动开发服务器

```bash
npm run dev
```

3. 生产构建

```bash
npm run build
```

## 目录结构

- `src/`
  - `components/` 站点通用组件
  - `data/` 模拟数据模型
  - `routes/` 页面视图
  - `index.css` 全局样式
  - `main.tsx` 应用入口

## 说明

该网站原型包含：首页、国家/地区数据页、行业市场页、趋势预测页、报告中心页、用户中心页和关于我们页，已实现产品需求文档中的主要模块展示与交互原型。

## 已覆盖的核心需求

- 全球经济仪表盘：核心指标卡片、每个指标趋势概览、预警展示
- 数据可视化分析：图表与卡片展示布局、交互式时间选择与指标对比入口
- 趋势预测与预警：预测卡片、当前预警列表与历史准确率展示
- 定制报告中心：报告模板选择、生成参数界面、在线报告入口
- 用户中心：个人信息、个性化配置入口、订阅设置说明
- 关于我们：品牌定位、平台说明、差异化策略

## 阿里云部署

该项目使用 `HashRouter`，适合部署在 Alibaba Cloud OSS 静态网站或其他静态托管服务。

部署步骤：

1. 运行构建命令：

```bash
npm run build
```

2. 将 `dist/` 目录上传到阿里云 OSS 静态网站托管目录。

3. 若使用 OSS CLI 上传：

```bash
ossutil cp -r dist/ oss://your-bucket-name/ --recursive
```

4. 在 OSS 控制台启用静态网站托管，设置首页为 `index.html`，错误页面也可设置为 `index.html`。

5. 访问生成的域名即可查看网站。
