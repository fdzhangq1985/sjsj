# 数据视界

这是一个按照仓库内 `需求文档` 生成的前端网站原型，基于 React + Vite 实现。

## 快速开始

1. 安装依赖

```bash
npm install
```

2. 启动开发服务器

```bash
npm run dev
```

3. 生产构建

```bash
npm run build
```

## 目录结构

- `src/`
  - `components/` 站点通用组件
  - `data/` 模拟数据模型
  - `routes/` 页面视图
  - `index.css` 全局样式
  - `main.tsx` 应用入口

## 说明

该网站原型包含：首页、国家/地区数据页、行业市场页、趋势预测页、报告中心页、用户中心页和关于我们页，符合产品需求文档的主要功能结构。
